# Finanzas Personales

## Descripcion del Proyecto

Plataforma web enfocada en ofrecer asesoramiento y herramientas para el manejo de finanzas personales, con especial enfasis en ayudar a personas que enfrentan problemas de endeudamiento. El proyecto busca proporcionar informacion clara y honesta para que los usuarios puedan tomar decisiones financieras informadas y recuperar su tranquilidad economica.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5.3.8
- Font Awesome 6.4.0

## Requisitos del Product Owner

Se confirma el cumplimiento del requisito del PO:

### Psicologia del Color

El proyecto implementa una estrategia consciente de psicologia del color para el bienestar de usuarios endeudados:

- **PROHIBIDO el uso del color ROJO** en numeros, graficos o elementos financieros, ya que genera ansiedad y miedo en personas con problemas de deuda.
- **Se utilizan colores verdes y azules** que transmiten calma, paz mental y confianza.
- Paleta de colores del proyecto:
  - #6bc3a3 (Verde agua)
  - #2a637a (Azul oscuro)
  - #3e8daf (Azul claro)
  - #8ee0c2 (Verde menta)
  - #000000 (Negro)

### Enfoque de Contenido - Sprint 1

El equipo confirma que entiende que el articulo para el Sprint 1 debe abordar directamente el tema de la **DEUDA**, que es el problema principal de nuestros usuarios objetivo. No se trata de ahorro general, sino de estrategias especificas para manejar y superar el endeudamiento.
